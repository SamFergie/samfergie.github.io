A professionally made game for laid back fun.

I used the RLNet and RogueSharp frameworks along with C# .NET to
create an experience to rival the classic PC game Rogue.

I made this project for my Computer Science A-Level coursework, for which I achieved 97% and an A*. I made use of agile methodologies and often set short, daily goals which helped keep me on task. By using C# .NET I made use of advanced programming frameworks such as LINQ statements when searching data structures. I made comprehensive use of object oriented programming which helped organise code and streamlin the debugging process.

What I learnt

During this project I learnt a lot about the trials of modern software development. Specifically, how the rigors of documentation can clash with the desire to keep working. I would often find myself coding hours into the night only to wake the next day with little understanding of what I'd done due to the lack of comments. By the end of the project I made sure all the code I made was thoroughly commented from the start.
